# mocha-chia-istanbul
JavaScript testing with Mocha Chia Istanbul

# Frist install 
##### $ yarn add global mocha
##### $ yarn add global chai
##### $ yarn add global istanbul

# test commands
## for mocha test only
#### $ mocha  
  ##### or 
#### $ yarn test 
  ##### or
#### $ nmp test

## mocha test with istanbul cover
#### $ yarn itcm
  ##### or 
#### $ istanbul cover _mocha
